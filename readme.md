# 💼 Card de Presentación - Bootstrap

Este proyecto es una card de presentación personal desarrollada con **HTML, CSS y Bootstrap**, como parte de un laboratorio práctico.

## 🚀 Tecnologías
- HTML5
- CSS3
- Bootstrap 5

## ✨ Características
- Diseño responsive
- Imagen de perfil con estilo personalizado
- Efectos hover y animaciones suaves
- Enlaces a redes profesionales

## 📌 Contenido
- Nombre y rol profesional
- Descripción breve
- Botones a LinkedIn, GitHub y portafolio
